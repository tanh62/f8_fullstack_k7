# dom

- Viết tắt của: Document Object Model
- Mô hình hoá đối tượng tài liệu HTML
-Tác dụng làm thay đổi HTML thông qua Javascript
```
Document:
-html document
    -head element:
        -title
    -body:
        -h1:
            -id
            -class
            -text
        -h2:
            -id
            -class
            -text
        -a:
            -href
            -target
            -title
            -text

```

## Các loại DOM


- DOM Document
- DOM Element
- DOM HTML
- DOM CSS
- DOM Event
- DOM Event Listerner
- DOM Node, NodeList
- DOM Navigation
